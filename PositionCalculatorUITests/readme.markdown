# Position Calculator 仓位计算器

一个简单的仓位计算器,帮助交易者快速计算合适的仓位大小。

## 功能特点

- 基于止损金额计算合适的仓位大小
- 自动计算1倍止盈价格
- 考虑手续费因素
- 支持快捷键操作
- 状态栏快速访问

## 使用说明

1. 输入开仓价格
2. 输入止损价格 
3. 输入可承受的止损金额(默认10)
4. 输入手续费率(默认0.1%)
5. 系统将自动计算:
   - 建议开仓金额
   - 1倍止盈价格
   - 手续费金额

## 快捷操作

- ESC: 关闭计算器窗口
- Enter: 在输入框中按回车进行计算
- 点击状态栏图标: 打开/关闭计算器
- 右键状态栏图标: 退出应用

## 注意事项

- 所有价格输入需为正数
- 手续费率以百分比形式输入(如0.1表示0.1%)
- 计算结果会考虑手续费成本
